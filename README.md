-----------------
# Compatible OS #
-----------------
	Mac OS
	Unix-like OS

----------------
# Description  #
----------------
Command line interface for finding books recommended by author in an ebook(epub or mobi type).

----------------
# Requirement #
----------------
	zipfile
	$ sudo pip3 install zipfile

----------------
# Usage #
----------------
    $ python3 Ebooksfinder ebook.epub 
    $ python3 Ebooksfinder ebook.mobi
